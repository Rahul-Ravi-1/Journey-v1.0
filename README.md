# Journey

A simple Next.js screen using your sky background.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). The home page shows your `Assets/Sky.png` as a full-screen background with a minimal "Journey" title overlay.

## Project structure

- `src/app/page.tsx` – Home screen with sky image
- `src/app/layout.tsx` – Root layout
- `public/Assets/Sky.png` – Copy of your sky asset (served by Next.js)
- `Assets/` – Your original assets (Sky.png, etc.)
